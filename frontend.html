QR Code Based Crowd Monitoring System

This project is used to control and monitor the number of people inside a place. When a person enters or exits, they scan a QR code at the gate. The system automatically counts how many people are inside and helps prevent overcrowding.

It is simple, fast, and useful for places like colleges, malls, offices, events, and public areas.
